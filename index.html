Quickstart

Call the API from any language that can send HTTPS.

# 1. Set your API key
export A2A_API_KEY="your-key"

# 2. Send a test event
curl -X POST https://api.a2ashield.cloud/v1/score \
  -H "Authorization: Bearer $A2A_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "event_type": "door_open",
    "location": "main-entrance",
    "time": "2025-11-15T14:22:00Z"
  }'

Endpoints

Core routes for scoring and telemetry.

POST /v1/score

Score a single event or log line. Returns risk band + numeric score.

POST /v1/batch

Send batches of events. Useful for log replay and backfill.

GET /v1/health

Simple health endpoint for your monitoring and status checks.

Event schema (v1)

Minimal, typed, and extensible.

{
  "event_type": "string",    // required
  "subject_id": "string",    // user, device, or system ID
  "ip": "string",
  "location": "string",
  "time": "ISO-8601 timestamp",
  "metadata": { "any": "json" }
}
